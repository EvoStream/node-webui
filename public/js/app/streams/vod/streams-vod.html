Video On Demand

The first critical step to getting Video on Demand (VOD) working is to place all of the media files into the appropriate directory.
By default that directory is the media folder in the main EMS folder. If a different folder is to be used for the media files, simply modify the config.lua file and change the mediaFolder parameter with the new path.
